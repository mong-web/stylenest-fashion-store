# 🛍️ StyleNest - Fashion Store Landing Page

StyleNest is a modern, responsive, and interactive F-Commerce fashion landing page designed to offer a seamless shopping experience. The platform features clean UI components, dark mode integration, dynamic product filtering, and direct social ordering capabilities.

🌐 **Live Demo:** [https://stylenest-fashion-store.netlify.app/](https://stylenest-fashion-store.netlify.app/)

---

## 📂 Project Architecture

The project follows a clean, modular structure for easy maintenance and scaling:

* **`index.html`** - Contains the semantic structure, layout, product galleries, modal views, and navigation components.
* **`style.css`** - Handles custom styling, visual themes, dark mode CSS variables, responsive breakpoints, and animations.
* **`script.js`** - Manages dynamic client-side interactions, theme toggling, product filtering logic, and quick view overlays.

---

## ✨ Features

* 🌙 **Dark Mode Toggle:** Seamless switching between Light and Dark themes.
* 📱 **Fully Responsive Layout:** Optimized for mobile, tablet, and desktop screens.
* 🔍 **Interactive Product Filter:** Easily browse products by category.
* 👁️ **Quick View Modal:** Inspect product details without navigating away from the page.
* 📲 **Direct Social Ordering:** Quick action buttons for ordering via social messaging.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic layout and page markup.
* **CSS3:** Custom styles, Flexbox/Grid, media queries, and root variables.
* **JavaScript (ES6):** DOM manipulation and interactive features.
